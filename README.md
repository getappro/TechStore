# TechStore

A modern e-commerce starter built with Next.js, React and TypeScript. This project is designed as a learning playground to help you understand how to build scalable e-commerce applications using current web standards.

## Objectifs pédagogiques
- Comprendre la structure d'un projet Next.js + TypeScript
- Apprendre à créer des pages, composants et styles
- Se familiariser avec les bonnes pratiques (scripts, linting, build)

## Installation
1. Clonez le repo :

   git clone https://github.com/getappro/TechStore.git
   cd TechStore

2. Installez les dépendances :

   npm install

3. Lancez le serveur de développement :

   npm run dev

Le projet sera accessible sur http://localhost:3000

## Structure du projet
- src/pages : pages Next.js (routes)
- src/components : composants réutilisables
- src/styles : styles globaux

## Prochaines étapes suggérées
- Ajouter une page produit dynamique
- Intégrer Stripe pour les paiements
- Ajouter un CMS Headless (Strapi, Contentful)
- Mettre en place des tests (Jest, React Testing Library)
